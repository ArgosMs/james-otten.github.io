About

About Me

I’m James Otten and this is my personal blog. I’m a Software Engineer in Minnesota with a strong interest in application and network security. In my spare time I enjoy finding vulnerabilities in web applications, auditing code, maintaining my home lab, and writing software.

Purpose

The purpose of this blog is to help others learn about application security. I plan to post as many write-ups as I can in a public setting in the hopes that this information will be useful to those looking at improve application security by writing more secure software and useful to those tasked with assessing application security.

Disclaimer

Opinions expressed on this website are solely my own and do not express the views or opinions of my employer.

Any code or techniques provided by or described in this website should be used at your own risk and should only be used within systems and networks that you have been given permission to test.
